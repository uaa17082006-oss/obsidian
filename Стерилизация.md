### [[Асептика. Антисептика. Порядок сбора, хранения, удаления отходов.#**Дезинфекция** - **не полное** уничтожение м/о на **неживых объектах** (поверхности, инструменты, помещение]].|Дезинфекция)
- физическая(кипячение в дист.воде+20г/л соды 15-30мин)
- химическая(стекло, некороззийного металла, полимеры, резины в ванночки дизинфектанта)

	Дизифектанты:
	- галоид-
	- кислород-
	- альдегид-
	- фенолсодержащие
	- ПАВ
	- гуанидины
	- спирты

### ПРЕДСТЕРИЛИЗАЦИОННАЯ ОЧИСТКА
- Промыть каждый инструмент 30сек в дез.растворе ёршиком
- Ополоснуть проточной водой 5-10мин
- Ополоснуть дист. водой 30сек

```
Механизированный способ, заменяющий 1,2 этапы - ультразвуковые ванночки наполненные Лизетолом АФ
```

### СУШКА
- В разобранном виде на чистой сухой салфетке
- Либо сухожар

### КОНТРОЛЬ ПРЕДСТЕРИЛИЗАЦИИ
Берут 10% изделий и делают пробы:
1. **Фенолфталеиновая.** 1г фенолфталеина+60г спирта+40г дист.воды. Розовый - тест на **моющие вещества** положительный
2. **Азопирамовая**. Азопирам + 3% перекись 1к1. Фиолетовое окрашивание - положительный тест. Бурое - ржавчина.
3. **Амидопириновая.** 5%спирт + 30% ук.к-ты + 3% перекись 1:1:1. Сине-зеленое окрашивание - тест на **кровь** положительный.
4. **Бензидиновая**. 0.025г бензидина + 1г 3%перекиси + 50мл 50% ук. к-ты. Синее окрашивание - тест на **кровь** положительный.

### Методы стерилизации:
1. Сухожар
2. Автоклав
	- в двух режимах: 130-20 и 120-45. 
	- в спец. бумаге(из бязи, высокопрочной, крепированной)
3. Гласперлен
4. Химический метод
	- 6% перекись 50градусов 3часа
5. Стерилизация газом
	- для колющего режущего
	- пары формальдегида


Купить пластилин на А4 послойно зуб выложить(эмаль, дентин, цемент, пульпа)
